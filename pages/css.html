Flexible Box (Flexbox)

Um guia completo sobre as propriedades do layout Flexbox no CSS

Flex-direction

flex-direction é um contêiner flexível, que define a direção e formato de um elemento ou conteúdo em torno do eixo principal e eixo transversal.


Chamamos o eixo principal de main-axis e o eixo transversal de cross-axis.

Para flex-direction seu valor padrão é flex-direction: row, definido sua direção para linha, de acordo com o eixo main-axis.


    .container {
        flex-direction: row;
    }

Além da direção em linha, também existe a direção em coluna, definido pelo valor flex-direction: column, de acordo com o eixo cross-axis ou eixo transversal.


    .container {
        flex-direction: column;
    }

Diferente do valor row, flex-direction: row-reverse tem seu valor invertido, começando a linha da direita para a esquerda.


    .container {
        flex-direction: row-reverse;
    }

Diferente do valor column, flex-direction: column-reverse tem seu valor invertido, começando a coluna de baixo para cima.


    .container {
        flex-direction: column-reverse;
    }

Flex-wrap

Em CSS flex-wrap é um contêiner flexível, definido se será forçado a ficar na mesma linha ou se irá quebrar em várias linhas, dependendo da quantidade de espaço disponível pelo elemento ou conteúdo.

flex-wrap têm por padrão o valor flex-wrap: nowrap, seu valor é definido para que o elemento ou conteúdo seja forçado a ficar na mesma linha.


    .container {
        flex-wrap: nowrap;
    }

flex-wrap: wrap têm por valor definido, para quando o elemento ou conteúdo não tenha espaço suficiente para crescer mais, será forçado a quebrar em várias linhas.


    .container {
        flex-wrap: wrap;
    }

flex-wrap: wrap-reverse ao invés de quebrar em várias linhas de cima para baixo como o padrão, será forçado a quebrar de baixo para cima.


    .container {
        flex-wrap: wrap-reverse;
    }

Flex-flow

Flex-flow é a junção de flex-direction com flex-wrap, ao invés de fazer duas declarações flex-derection: row; e flex-wrap: nowrap; simplificando com apenas uma declaração flex-flow: row nowrap;


    .container {
        flex-flow: row nowrap;
        flex-flow: row wrap;
        flex-flow: row-reverse nowrap;
        flex-flow: row-reverse wrap;
        flex-flow: row wrap-reverse;
        flex-flow: row-reverse wrap-reverse;
    }
    


    .container {
        flex-flow: column nowrap;
        flex-flow: column wrap;
        flex-flow: column-reverse nowrap
        flex-flow: column-reverse wrap
        flex-flow: column wrap-reverse
        flex-flow: column-reverse wrap-reverse
    }

Justify Content

Center

Centraliza os itens na horizontal.

Space-between

Cola o primeiro item no começo do eixo e o último item no final do eixo e calcula o espaço entre eles automaticamente.

Space-evenly

Diferente do space-between, o space-evenly não cola o primeiro e o último item no começo e final do eixo, apenas calcula o espaço entre eles automaticamente.

Align-content

Alinha os elemntos no eixo transversal, somente quando estão empacotados. Os valores aceitos são:

Align-content: stretch

Align-conten: stretch; tem por padrão dividirá a área disponível, linhas ou colunas, em partes iguais e irá esticar os elementos do tamanho do cross-axis ou main-axis

Align-content: flex-start

Align-content: flex-start; Irá pegar todos os elementos e grudar no croos-start e reservará o espaço em branco em baixo.

Align-content: flex-end

Align-content: flex-start; Irá pegar todos os elementos e grudar no croos-start e reservará o espaço em branco em baixo.

Align-content: center

Align-content: center; alinhará os elementos no meio do cross-axis.

Align-content: space-between

Align-content: sparce-between; os primeiros elementos ficarão grudados no cross-start e os últimos elementos ficarão grudados no cross-end e ficará um espaço em branco no meio do cross-axis.

Align-content: space-evenly

Align-content: space-evenly; coloca um espaçamento antes, entre e depois dos elementos.

Align-content: space-around

align-content: space-around; dividirá o cross-axis em partes, sendo a primeira e a última metade dos espaços entres elementos e colocará os elementos centralizados.

Order

Todo item em flexbox tem por padrão order: 0.

Caso a order tiver valores difentes de 0, o css formatará os itens em ordem crescente de acordo com seus valores.

Align-self

Aplicado a cada um dos itens dentro do flexbox de acordo com o alinhamento de cada item no eixo transversal. Os alinhamentos são:

Align-self: auto

Herdar a característica de alinhamento vertical(align-content ou align-items) de seu pai.

Align-self: flex-start

Alinhamento grudado do cross-start.

Align-self: flex-end

Alinhamento grudado do cross-end.

Align-self: center

Alinhamento no centro entre o cross-start e o cross-end.

Align-self: stretch

Alinhamento ocupando todo o espaço dividindo a área disponível, linhas ou colunas, em partes iguais e irá esticar os elementos do tamanho do cross-axis ou main-axis.

Flex-basis

Flex-basis é um parâmetro para alinhamento de itens e não de contêiners, e têm os seguintes alinhamentos:

Em CSS flex-basis tem por padrão, valor auto. Significa que o tamanho do elemento no eixo principal, altura ou largura, é ditada pelo tamanho de seu conteúdo.


    .item {
        flex-basis: auto;
    }

Definindo um valor para flex-basis em px, cada elemento terá seu tamanho definido de acordo com o valor, independente do tamanho do conteúdo, porém o tamanho em px, ficará alinhado enquanto o contêiner conseguir suportar.


Ou seja, se o contêiner tiver 300px, tiver 3 elementos e cada elemento tiver 150px, cada elemento dentro desse contêiner terá seu tamanho alinhado de acordo com o espaço suportado, ou seja, diminuirá para 100px cada um.

    
        .item {
            flex-basis: 300px;
        }

Flex-grow

Flex-grow tem por padrão o valor 0. Definindo o quanto esse elemento aumentará em tamanho.

Flex-shrink

Flex-shrink tem por padrão o valor 1. Definindo o quanto esse elemento diminuirá em tamanho.

Propriedade Flex

A propriedade "flex" é a junção de flex-basis, flex-grow e flex-shrink em um único comando, definido os valores sempre na seguinte ordem: flex-grow, flex-shrink e flex-basis.

Formatações flex de substituição

flex: 0 1 auto
flex: initial

flex: 0 0 auto
flex: none

flex: 1 1 auto
flex: auto

flex: 3 1 auto
flex: 3
